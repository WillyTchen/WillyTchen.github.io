Unit 3

Throughout this unit, we have learned how to link your HTML file to an external style sheet, change the color of HTML elements in CSS, change the font of elements in CSS, apply the box model to organize our web pages, use ID and Class to distinguish certain elements, and learn how to layout our page properly.

The most important topic that we learned in this unit was distinguishing between padding, border, and margin.

Here is what I have learned in this unit:

  • Linking an external style sheet to an HTML file
  • Changing the font-family of an element
  • Using padding, border, and margin to organize our website
  • Using ID and Class to select certain elements
  • Using relative, absolute, or fixed positioning

  • Creating a button
  • Creating a web page banner
